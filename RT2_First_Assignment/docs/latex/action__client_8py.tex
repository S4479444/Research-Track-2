This node implements the robot\textquotesingle{}s action client.

This node implements both the graphical user interface (G\+UI) and the action client node of the robot. In particular, the action client handles the behavour of the robot, based on the user\textquotesingle{}s input, between choosing a specified goal position (x,y), or canceling any previous command and stopping the robot.
